# calculadora-rocket
Terminal de Cálculo de Procura Técnica - Rocket Enduro Parts. Aplicación Web Progresiva (PWA) para la auditoría de costos, gestión de márgenes y validación de compatibilidad técnica OEM/Aftermarket. Optimizada para el ecosistema de motocicletas de alto rendimiento y cumplimiento de estándares de integridad en la cadena de suministro.
